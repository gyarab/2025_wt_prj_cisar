# Hockey Base
Projekt HockeyBase představuje webovou <u>databázovou aplikaci</u> zaměřenou na evidenci a prezentaci <u>dat</u> ze světa <u>ledního hokeje</u>. Cílem aplikace je vytvořit přehledný <u>informační systém</u>, který umožní <u>uživatelům</u> zobrazovat informace o <u>týmech</u>, <u>hráčích</u> a jednotlivých <u>zápasech</u> <u>české reprezentace</u> a <u>Tipsport extraligy</u>. Aplikace využívá strukturovanou <u>databázi</u>, ve které jsou ukládána <u>sportovní data</u> a <u>statistiky</u>.

Uživatel se po vstupu do aplikace dostane na <u>domovskou stránku</u>, kde se nachází nejnovější <u>články</u> o aktuálním dění ve světě hokeje. Z domovské stránky nebo-li <u>Homepage</u> může přejít do hlavních <u>sekcí</u> aplikace. První z nich je sekce <u>Týmy</u>, která obsahuje <u>seznam</u> všech týmů extraligy seřazených v aktuální <u>tabulce</u> ligy. Uživatel si může každý tým rozkliknout kde se mu zobrazí základní <u>informace</u> o daném týmu mj. <u>klubové barvy</u>, <u>stadion</u>, <u>soupiska</u> nebo poslední odehrané zápasy. Další sekcí jsou <u>Hráči</u>, kde se nachází seznam všech hráčů, ve kterém se dá pohodlně vyhledávat. U každého hráče se po rozkliknutí objeví jeho <u>statistika</u> – <u>pozice</u>, <u>klub</u>, <u>góly</u> a <u>body</u>.

Poslední sekcí aplikace jsou <u>Zápasy</u>. Ta obsahuje <u>přehled</u> všech odehraných <u>utkání</u> v aktuální <u>sezoně</u> v extralize a zápasů české reprezentace. U každého zápasu se dá zobrazit <u>detail</u>, kde se udává výsledné <u>skóre</u>, <u>střelci gólů</u>, <u>střely</u> a <u>fauly</u>.

Aplikace má též přístup z <u>funkce administrátora</u>. <u>Administrátor</u> má přístup do <u>sekce administrace</u>, kde může provádět <u>správu databáze</u>. V této části může přidávat nové týmy, hráče nebo zápasy a upravovat existující informace a statistiky.
